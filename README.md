# eslint-config-noclocks

![NPM Version](https://img.shields.io/npm/v/@noclocksdev/eslint-config-noclocks)


## Installation

```bash
npx install-peerdeps --dev eslint-config-noclocks
```

## Usage

In your `.eslintrc.js`:

```tsx
module.exports = {
  extends: ['mantine'],
  parserOptions: {
    project: './tsconfig.json',
  },
};
```
